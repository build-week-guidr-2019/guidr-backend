## Auth Endpoints
`/api/login`--> Logins
`/api/register`--> Registers

## Guides/Users Endpoints 
`/api/guides` --> post
    `/:id` -->the guide id, put, get 
    `/:id/trips`--> guide's trips (gets trips)


## Trips
`/api/trips` --> post, get
`/api/trips/:id`-->get, put, delete

## Types
`/api/types` --> GET, POST
`/api/types/:id` --> GET
